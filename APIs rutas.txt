Login

POST http://localhost:9000/api/auth/register
POST http://localhost:9000/api/auth/login (Genera token para el usuario)
GET http://localhost:8080/api/auth/me (Necesita el token generado en Login)

Mascotas

Registrar - POST http://localhost:9000/api/mascotas
Listar - GET http://localhost:9000/api/mascotas
Obtener - GET http://localhost:9000/api/mascotas/1
Actualizar - PUT http://localhost:9000/api/mascotas/1
Eliminar - DELETE http://localhost:9000/api/mascotas/1

Eventos

Registrar - POST http://localhost:9000/api/eventos
Listar - GET http://localhost:9000/api/eventos
Obtener - GET http://localhost:9000/api/eventos/1
Actualizar - PUT http://localhost:9000/api/eventos/1
Eliminar - DELETE http://localhost:9000/api/eventos/1
